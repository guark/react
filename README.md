# Guark React Template
Guark react app template.


## Install

Create new empty directory and cd to it, and run:
```bash
guark init --template react --mod github.com/username/appname
```

## Run dev app.

Run dev app with vue hot reload:
```bash
guark run
```

## Build

```bash
guark build
```

## Bundle

```bash
guark bundle
```
